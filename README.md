# Portfólio

 No projeto desenvolvi um layout inicial de um portfólio de desenvolvimento de software. A ideia desses projeto era exercitar os conceitos que aprendi de HTML e CSS aplicando num site real que poderia ser utilizado mais a frente como portfólio.

### Tecnologias Utilizadas

* [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element)
* [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS)

## Como rodar os testes

O projeto final pode ser visualizados em: 
https://stecarvalho87.github.io/portfolio/

## ⚠️ Problemas enfrentados

### Problema 1:
Na execução do projeto tive dificuldades em editar a imagem que é muito larga para a página.
* Solução encontrada: Defini o tamanho máximo no css e utilizei a tag "object-fit: cover.
